# File Operations

All file operations are available on both `AsyncKCFinderClient` (async) and `KCFinderClient` (sync). The examples below show async usage first, with the sync equivalent immediately after.

Directories in KCFinder are relative to the configured `uploadDir`. For example, if `uploadDir` is `files/` and you want to work in `files/images/banners/`, the `dir` argument is `"images/banners"`.

## list_files

List all files in a directory. Returns a list of `FileInfo` objects.

```python
# Async
files = await client.list_files("images/banners")

# Sync
files = client.list_files("images/banners")

for f in files:
    print(f.name)         # "banner_2024.jpg"
    print(f.size)         # 204800 (bytes)
    print(f.mtime)        # datetime(2024, 3, 15, 10, 30, tzinfo=timezone.utc)
    print(f.is_writable)  # True
```

### FileInfo Fields

| Field | Type | Description |
|-------|------|-------------|
| `name` | `str` | Filename (no path) |
| `size` | `int` | File size in bytes |
| `mtime` | `datetime` | Last modified time (UTC) |
| `is_writable` | `bool` | Whether the file can be modified |

## upload

Upload one file or a list of files to a directory.

```python
from pathlib import Path

# Upload a single file
await client.upload("images/banners", Path("./banner_spring.jpg"))
client.upload("images/banners", Path("./banner_spring.jpg"))

# Upload multiple files at once
await client.upload("images/banners", [
    Path("./banner_spring.jpg"),
    Path("./banner_summer.jpg"),
    Path("./banner_fall.jpg"),
])
client.upload("images/banners", [
    Path("./banner_spring.jpg"),
    Path("./banner_summer.jpg"),
    Path("./banner_fall.jpg"),
])
```

Returns `None` on success. Raises `UploadError` if any file fails to upload.

## download

Download a file and return its content as bytes.

```python
# Async
content = await client.download("images/banners", "banner_spring.jpg")

# Sync
content = client.download("images/banners", "banner_spring.jpg")

# Save to disk
with open("downloaded.jpg", "wb") as f:
    f.write(content)
```

## delete

Delete a file.

```python
# Async
await client.delete("images/banners", "old_banner.jpg")

# Sync
client.delete("images/banners", "old_banner.jpg")
```

Returns `None` on success. Raises `FileOperationError` if the file does not exist or cannot be deleted.

## rename

Rename a file in place (cannot move to another directory).

```python
# Async
await client.rename("images/banners", "banner_v1.jpg", "banner_v2.jpg")

# Sync
client.rename("images/banners", "banner_v1.jpg", "banner_v2.jpg")
```

Parameters:
- `dir` — the directory containing the file
- `file` — current filename
- `new_name` — new filename (no path)

## get_thumbnail

Retrieve the KCFinder-generated thumbnail for an image as PNG bytes.

```python
# Async
thumb = await client.get_thumbnail("images/banners", "banner_spring.jpg")

# Sync
thumb = client.get_thumbnail("images/banners", "banner_spring.jpg")

# Save thumbnail to disk
with open("thumb.png", "wb") as f:
    f.write(thumb)
```

Thumbnails are generated by KCFinder on the server side. The returned bytes are always PNG.

## Error Handling

```python
from kcfinder_client import FileOperationError, UploadError, PermissionDeniedError

try:
    await client.delete("images/banners", "protected.jpg")
except PermissionDeniedError as e:
    print(f"No permission: {e.message}")
except FileOperationError as e:
    print(f"File operation failed ({e.action}): {e.message}")
```

See [exceptions.md](exceptions.md) for the full error hierarchy.
